
# Vue Data Dashboard

## Overview
This project aims to create a data dashboard using Vue.js for the frontend and Node.js/Express for the backend.

## How to run
### Frontend
- `cd frontend`
- `npm install`
- `npm run serve`

### Backend
- `cd backend`
- `npm install`
- `npm start`
